1.概要
	テンキーや数字キーで好きなフォルダに移動・コピーするソフトです。
	各フォルダにあった名前に、番号を付けて転送することも可能です。
	対応形式はJPG、BMP、GIF、PNGです。
	ソフト名は「サイアス」とお読みください。

2.動作環境
	Windows2000で動作確認
	VB6ランタイムが必要です。
	ActiveXエラーが出る場合はWindowsUpdateでIE等を更新してください。

3.インストール
	ダウンロードしたファイルを好みのフォルダに解凍、展開。
	必要に応じてショートカットを作成してください。

4.使用・操作方法

	● 4.1 基本的な操作
	簡易マニュアルの方をご参照ください

	● 4.2 操作方法

	【キーボード】
	数字キー … 割り当てたフォルダに転送 (Ctrlを押しながらだとそのフォルダを開く)
	+        … 次のページ
	-        … 前のページ
	Space    … フォルダ内の次ファイルを開く (Shiftを押しながらだと前ファイル)
	BkSpace  … フォルダ内の前ファイルを開く
	Delete   … ファイル削除

	Enter    … 画像枠内伸長
	方向キー … 上下でページ切り替え、左右で表示画像変更(シフトを押しながらだと画像のスライド)
                 (オプションで画像のスライドに変更可能)
	C        … コピーモードに変更
	M        … 移動モードに変更
	L        … 転送・削除操作のロック (Ctrl+数字の移動の際、誤って転送しないためのロック)

	【マウス】
	フォルダ一覧左クリック … 指定のフォルダに表示画像を転送
	フォルダ一覧右クリック … フォルダの設定ウィンドウを開く
	フォルダ一覧ホイールクリック … 指定のフォルダを開く
    フォルダ一覧へドラッグ＆ドロップ … ドロップしたフォルダを登録する

	画像ドラッグ   … 画像スライド
	画像左クリック … 次のファイル
	画像右クリック .. 前のファイル

	ページ一覧左クリック … 指定のページを開く
	ページ一覧右クリック … ページ名の変更

	【機能ボタン】
	画像削除 … 表示されている画像ファイルを削除します
	引き伸ばし … 表示されている画像を枠に合わせて拡大縮小します
	転送切り替え … 転送モードを「移動」「コピー」と交互に切り替えます。
	転送ロック … 転送と削除が出来ないようにします。もう一度押すと解除します。


	● 4.3 フォルダ登録
	・転送先フォルダの登録です。フォルダの登録は最大10x250個
	・ページ内フォルダ一覧を右クリックすると設定が可能です
	・フォルダ一覧にエクスプローラ等からドラッグアンドドロップすると登録が簡単です

	● 4.4 ページ編集
	・フォルダを最大10個登録できるのがページです。ページは250個まで登録できます。
	・「▲」「▼」ボタンで選択中のページを上下にずらすことができます。
	・ページを増やすには「新規」を、選択中のページを削除するには「削除」を押してください。

	● 4.5 個別連番転送
	・各フォルダに決められたファイル名に番号を沿えて転送する設定です。
	  背景_021.jpg といったファイル名に変更して転送することができます。
	・フォルダ設定ウィンドウ内の個別連番転送名欄の左のチェックボックスで可否を設定します。
	・固定名は 背景_021.jpg の 「背景」に当たる部分です。
	・次転送番号はオプションの「個別連番転送の最後の番号を記憶」の有効時だけ機能します。
	  最後に転送したファイル 背景_021.jpg の「021」の番号の次の番号を記憶します。
	  欠番を無視して番号を付けたい人はどうぞ。
	・番号の桁数や、転送名との間の文字についてはオプションの連番転送設定で変更できます。


	● 4.6 オプション

	■ 転送設定

	  □ 同名ファイルの処理
	    ・上書きの問い合わせダイアログを表示  … 上書きかファイル名変更(連番付け可)か問い合わせます。
	    ・強制的に上書き                      … 確認無しに上書きをします
	    ・末尾に連番を付けて保存              … ファイル名の後ろに番号を付けて転送します

	  □ 連番転送設定
	    接続文字     … 「背景_021.jpg」の _ の部分を設定します
	    0の桁数      …  設定桁数に満たない位を0で埋めます。6にすると「背景_000021.jpg」になります。
	    連番開始番号 … 番号を何番から始めるか設定します。2にすると「背景_002.jpg」から始まります。
	    個別連番転送の最後の番号を記憶 … 個別連番転送設定時、最後に転送した番号の次を記憶します。

	■ 削除処理

	  □ 削除時の動作
	    ごみ箱の設定に準拠 … ファイルをごみ箱に移します。ごみ箱を使用しない設定だと直接削除します。
	    直接削除           … ごみ箱を使わず、直接削除します
	
	  □ 削除警告
	    なし … 削除警告をせずキーが押されたらファイルを削除します
	    あり … 削除警告を出して問い合わせます

	■ 操作設定

	  □ カーソルキーの設定
	    画像のスクロール … 大きな画像をスクロールします。シフトキーを押しながらで3倍速になります。
	    ↑↓：ページ移動、←→：画像選択 … 上下キーでグループの移動、左右キーでファイルを移動します。

	● 4.7 注意事項
	インターフェースの変更のため、現時点ではボタンがアクティブになっていると(フォーカスがある状態)
	キーボード操作がWindows標準のものとなり、キーボード操作方法に書かれているキーボード操作ができません。


5アンインストール
